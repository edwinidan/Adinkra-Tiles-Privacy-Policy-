# Walkthrough

This repository is a static privacy-policy website for Adinkra Tiles.

Publicly served files live in `public/`:

- `public/index.html`
- `public/styles.css`
- `public/adinkra tiles logo.png`
- `public/app-ads.txt`

The site does not use Vite, Next.js, or any other build system. Vercel serves the static files directly from `public/`.
